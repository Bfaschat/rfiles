# RFiles Bot

Demo [@rfilesbot](https://telegram.me/rfilesbot)

## Requirements
* PHP 5.2+


## Installation
* edit config.php
* open via browser `https://api.telegram.org/bot<BOT TOKEN>/setWebhook?=https://yourdomain.com/rfiles.php`
